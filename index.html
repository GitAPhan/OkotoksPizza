Content tags are great, but we need a way to split up our websites into logical divisions so we can use layout rules to get things is just the right position for our designs.

Copy the written content from the Okotoks Pizza project description and put it into your website where it makes the most sense

You can also add your own extra content if you would like! Split your layout elements and written content in to containers:

  • div
  • article
  • section
  • header
  • footer
  • etc.
Pick the layout element that makes the most sense for the section. Once complete, add, commit, and push the files with the commit message "Added layout tags". Submit the GitHub repository link for a submission.
